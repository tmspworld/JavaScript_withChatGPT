JavaScript Tutorial for React Learners

Understanding JavaScript is essential before diving into React. This tutorial covers fundamental JavaScript concepts that React developers should be familiar with.

1. Variables and Constants

In JavaScript, we use var, let, and const to declare variables.


let name = 'React Learner'; // let allows the value to be changed
const appName = 'React';    // const value cannot be changed

Example:


let age = 25;
const pi = 3.14;

age = 26; // valid
pi = 3.15; // Error: Assignment to constant variable

2. Arrow Functions

Arrow functions are a shorter way to write functions in JavaScript. They are commonly used in React for defining functional components and callbacks.


const greet = () => {
  console.log('Hello, welcome to React!');
};

greet(); // Outputs: Hello, welcome to React!

3. Destructuring

Destructuring allows you to extract values from arrays or properties from objects into distinct variables.


// Example with Arrays:
const numbers = [1, 2, 3];
const [first, second, third] = numbers;

// Example with Objects:
const user = { name: 'Mandeep', age: 25 };
const { name, age } = user;

4. Spread and Rest Operator

Spread is used to expand an array or object, while Rest collects remaining elements into an array.


// Spread Example
const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5];

// Rest Example
const sum = (...args) => args.reduce((total, current) => total + current, 0);

5. Array Methods

JavaScript provides various methods for arrays that are frequently used in React.


// map() Method Example
const numbers = [1, 2, 3];
const doubled = numbers.map((num) => num * 2);

// filter() Method Example
const evenNumbers = numbers.filter((num) => num % 2 === 0);

6. Template Literals

Template literals allow you to create strings with embedded expressions. Use backticks (`) instead of single or double quotes.


const name = 'React Learner';
const greeting = `Hello, ${name}!`;

7. Promises and Async/Await

JavaScript promises handle asynchronous code. With async/await, you can handle promises more cleanly.


// Promise Example
const fetchData = () => {
  return new Promise((resolve) => {
    setTimeout(() => resolve('Data loaded'), 1000);
  });
};

fetchData().then((message) => console.log(message));

// Async/Await Example
const fetchData = async () => {
  const result = await new Promise((resolve) => {
    setTimeout(() => resolve('Data loaded'), 1000);
  });
  console.log(result);
};

8. Classes and Objects

JavaScript classes are used to create objects. In React, components can be written as classes, though functional components are now more common.


class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    console.log(`Hello, my name is ${this.name}`);
  }
}

const mandeep = new Person('Mandeep', 25);
mandeep.greet(); // Hello, my name is Mandeep

9. Modules

JavaScript modules allow you to organize code into separate files.


// Exporting a Module
export const add = (a, b) => a + b;

// Importing a Module
import { add } from './math';

10. Ternary Operator

The ternary operator is a shorthand for if-else conditions, often used in React JSX for conditional rendering.


const isLoggedIn = true;
const message = isLoggedIn ? 'Welcome back!' : 'Please sign in';

These JavaScript concepts are vital when working with React. Understanding them will make your React development smoother and help you write cleaner code.
