JavaScript Tutorial for React Learners

Understanding JavaScript is essential before diving into React. This tutorial covers fundamental JavaScript concepts that React developers should be familiar with.

1. Variables and Constants

In JavaScript, we use var, let, and const to declare variables.


// JavaScript Code
let name = 'React Learner'; // let allows the value to be changed
const appName = 'React';    // const value cannot be changed

- Use let for variables that may change.

- Use const for variables that shouldnâ€™t change.

Example:


// JavaScript Code
let age = 25;
const pi = 3.14;

age = 26; // valid
pi = 3.15; // Error: Assignment to constant variable

2. Arrow Functions

Arrow functions are a shorter way to write functions in JavaScript. They are commonly used in React for defining functional components and callbacks.


// JavaScript Code
const greet = () => {
  console.log('Hello, welcome to React!');
};

greet(); // Outputs: Hello, welcome to React!

3. Destructuring

Destructuring allows you to extract values from arrays or properties from objects and assign them to variables.


// Destructuring arrays
const [first, second] = [10, 20];

// Destructuring objects
const {name, age} = {name: 'John', age: 30};

4. Spread & Rest Operator

The spread operator (...) allows an iterable (like an array) to expand, while the rest operator collects all remaining elements into an array.


// Spread Operator Example
const arr = [1, 2, 3];
const newArr = [...arr, 4, 5];

// Rest Operator Example
const sum = (...numbers) => numbers.reduce((acc, curr) => acc + curr, 0);

5. Array Methods

Commonly used array methods include map(), filter(), and reduce().


// Array.map()
const arr = [1, 2, 3];
const doubled = arr.map(num => num * 2);

// Array.filter()
const evenNumbers = arr.filter(num => num % 2 === 0);

// Array.reduce()
const sum = arr.reduce((total, num) => total + num, 0);

6. Template Literals

Template literals allow for easier string interpolation and multi-line strings.


const name = 'React';
const message = `Welcome to ${name} learning!`;

console.log(message); // Output: Welcome to React learning!

7. Promises & Async/Await

Promises and async/await make handling asynchronous operations more manageable in JavaScript.


// Using Promises
const getData = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve('Data received!'), 2000);
    });
};

getData().then(response => console.log(response));

// Using async/await
const fetchData = async () => {
    const result = await getData();
    console.log(result);
};

fetchData();

8. Classes & Objects

Classes in JavaScript are syntactic sugar over the prototypical inheritance model and allow you to create objects easily.


// Class Declaration
class Animal {
    constructor(name, species) {
        this.name = name;
        this.species = species;
    }
    speak() {
        console.log(`${this.name} makes a sound.`);
    }
}

const dog = new Animal('Dog', 'Canine');
dog.speak(); // Output: Dog makes a sound.

9. Modules

Modules allow you to split code into reusable pieces that can be imported and exported in different files.


// Exporting a function
export const sayHello = () => {
    console.log('Hello, React learner!');
};

// Importing a function
import { sayHello } from './module.js';

sayHello(); // Output: Hello, React learner!

10. Ternary Operator

The ternary operator provides a concise way to handle conditional expressions in JavaScript.


const isReactUser = true;
const message = isReactUser ? 'Welcome to React!' : 'Learn JavaScript first';

console.log(message); // Output: Welcome to React!

These JavaScript concepts are vital when working with React. Understanding them will make your React development smoother and help you write cleaner code.
